FORMATO DAS PERGUNTAS: UMA HISTÓRIA COM PERGUNTAS QUE O ALUNO INSERE A SUA OPÇÃO

- SERVIÇO JAVA WEB
- BANCO DE DADOS: MY SQL
- RELATORIOS: IREPORT
- IDE: ECLIPSE
- LINGUAGEM: JAVA
- CONTROLE DE VERSÃO: GIT HU B ???
- SERVIDOR DE IMAGEM: POSTIMAG.ORG

- PADROES DE PROJETO: MVC
	- MODEL: BANCO DE DADOS + PROGRAMAÇÃO DO BD.
	- VIEW: JSON (CONSUMIDO PELA APLICACAO WEB).
	- CONTROL: ROTAS HTTP.
- USUARIOS: ALUNO, CARA QUE INSERE AS PERGUNTAS (ADM) E O PSICOLOGO.


REQUISITOS:

- CADASTRO DO ALUNO: RA, CURSO, NOME, USUARIO, SENHA
- ALUNO SE LOGA NO SISTEMA POIS PRECISA SALVAR SUAS RESPOSTAS.
- SE ELE NÃO COMECOU, COMEÇA DO COMEÇO.
- SE O ALUNO JA COMECOU A REALIZAR O TESTE, O SISTEMA ABRIRIA NA QUESTÃO EM QUE PAROU A ULTIMA VEZ.
- CADA RESPOSTA (LETRA SELECIONADA) SOMA UMA PONTUAÇÃO EM DETERMINADA DISCIPLINA A SER AVALIADA (TRABALHO EM EQUIPE...) AO FINAL DO CURSO COM BASE NAS ESTATISTICAS É FORMADO UM PERFIL DE CADA ALUNO.
- AO FINALIZAR O QUESTIONARIO ELE EMITIRÁ UM RELATÓRIO PARA A PSICOLOGA, E TERMINARA COM UM OBRIGADO PARA O ALUNO.
- NO FINAL A PSICOLOGA GERA O RELATÓRIO COM O PERFIL E ENVIA PARA O ALUNO (E-MAIL).
- A PSICOLOGA PODE OU NÃO TECER ALGUM COMENTARIO SOBRE O PERFIL DE CADA ALUNO.
- SE O USUARIO TERMINAR O JOGO ELE NÃO JOGA MAIS NADA!!
